Visual Variables

Examples

  • Hue and Value Categorical
  • Hue and Value Diverging
  • Classic Multivariate Election Map
  • Size and Color Playground (Smart Mapping)
  • Winner-Take-All Occlusion

Principles

  • Data that will be compared across categories in a winner-take-all approach must be normalized
  • Polygons often a poor way to visualize unrelated quauntities like population distributions (think County-level election returns)

Principles

  • You don't need to choose between categorical and quantitative maps
  • Go for contrast
  • Same visual variables can be applied to charts (bubble-scatterplot, etc)
